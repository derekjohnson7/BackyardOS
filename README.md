# BackyardOS 
BackyardOS is an IoT evironmental monitoring system for my backyard to 
better understand the microclimateof my property.

The project began after repeadily losing my plants to suspected root rot 
casued by poor drainage in heavy clay soil. Rather than speculating and 
continuing to lose plants, BackyardOS collects environmental data using 
wireless sensors to monitor soil conditions, weather, and plant health 
over time.

## Goals
 - Monitor soil moisture across multiple garden beds.
 - Collect weather and environmental data.
 - Store historical sensor data.
 - Visualize trends through an interactive dashboard. 
 - Generate insights into drainage and plant health.
 - Provide a foundation for future automation and AI-driven 
recommendations.

## Planned Features
 - ESP32 Sensor Stations
 - Soil Mositure Moinitoring
 - Temperature/Humidity Sensing
 - Historical Database
 - Interactive Dashboard
 - Weather API integration
 - Camera Integration
 - Smart Irrigation
 - AI-powered Plant Health Insights
